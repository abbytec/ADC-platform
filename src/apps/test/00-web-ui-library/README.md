# Web UI Library

Web Components (Stencil) para el namespace `default` (desarrollo).

## Uso
```typescript
import "@ui-library"; // Auto-registra componentes
import "@ui-library/styles"; // CSS base
import "./styles/tailwind.css"; // Extensiones locales
```

## Componentes
`<adc-button>`, `<adc-header>`, `<adc-stat-card>`, `<adc-error>`, etc.
